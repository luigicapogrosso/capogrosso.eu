# capogrosso.eu

Personal website built with plain HTML and CSS.

## Overview

This repository contains the source for a simple static website served from `index.html` with styling in `css/style.css`.
Assets and documentation are stored in the `img/` and `docs/` folders respectively.

## Deploy

This is a static site and can be hosted on GitHub Pages, or any static hosting provider.
For GitHub Pages, enable Pages in the repository settings and point it to the `main` (or `gh-pages`) branch root.
